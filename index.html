Bootstrap


Bootstrap é um conjunto de ferramentas (framework) que auxilia a construção de layouts para aplicações web. Consiste em vários templates de html e CSS/LESS com componentes como botões, widgets de formulários, menus e outros. Uma qualidade do framework é a sua compatibilidade com aplicações web (ver design responsivo). Criado por desenvolvedores do Twitter em 2011, hoje é um dos recursos mais populares para criar layouts web. Sites populares como o globo.com usam Bootstrap.

HTML5


O HTML (HyperText Markup Language) é a linguagem da web, desde o seu início em 1993. Ela define a estrutura da página (enquanto o CSS a configura seu visual). Com o passar dos anos, ela foi se atualizando até a alcançar a versão 5, que trouxe mudanças consideráveis Seu logotipo mostra do que ela se trata, o HTML5 veio para empoderar a web! Novidades como o uso de tags semânticas que deixam o código mais legível tanto para buscadores como o Google (SEO) quanto humanos. Recursos adicionais como <audio>, <vídeo> e <canvas> para desenhos (adeus Flash) são o que eriquecem a web hoje em dia.

CSS3


Junto com o html5, o CSS (Cascading Style Sheets) é a linguagem que define todo aspecto visual (layout, cores, tipografia) das página web. Chega sua versão 3 com recursos novos que tornam a web mais bonita, como uso de degradês, sombras, bordas arredondadas e animações nos elementos, Além uma sintaxe de seletores mais rica e aprimorada.

LESS


O CSS pode ter evoluido, mas ele continua sendo uma linguagem estática. Isso quer dizer que não se pode usar recursos que linguagens de programação (variáveis, objetos, funçnoes) usam para dar dinamismo e facilitar o desenvolvimento. Para isso foi criado o LESS, uma linguagem dinâmica que pode ser compilada em CSS. Dentre seus principais recursos pode-se citar o uso de variáveis (muito útil para cores, por exemplo), aninhamento de estilos (imagem), mixins (propriedades combinadas), funções, operações e namespaces que tornam o trabalho do editor de CSS tão dinâmico e inteligente quanto o de um programador.

Photoshop & Illustrator


Existem diversas ferramentas para criação web, mas para criar imagens e desenhos as melhores escolhas continuam sendo o Photoshop e o Illustrator. Photoshop é o editor profissional de fotos e imagens raster, Illustrator edita imagens vetoriais. Hoje na versão CC (Creative Suite) são tantas as funcionalidades que extenderia muito o texto, mas os dois trabalham sobre layers, manipulam cores, possuem filtros e exportam em vários formatos.

Design Responsivo


Há alguns anos atrás, os desenvolvedores de sites faziam um layout para computador em uma página outro para celular em um link diferente. Com a ascensão dos smartphones e de monitores com telas cada vez maiores (full HD) isso precisou mudar. O Design Responsivo consiste em técnicas e recomendações para fazer o layout da página se adaptar à tela em que é apresentada. Seus princípios são o uso de layout fluido (sem medidas absolutas), o uso do media-query do CSS3 (@media), e imagens e texto com tamanhos adaptáveis. Esta página usa Bootstrap e consegue se adaptar a diferentes mídias.
